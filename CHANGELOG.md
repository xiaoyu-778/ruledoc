# 更新日志

所有 notable 的更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2026-04-20

### Added

- 初始版本发布
- 全新架构设计，规则与核心完全解耦
- 支持 Markdown、DOCX 输入格式
- 支持扬州大学学位论文格式 (yzu_thesis)
- 支持扬州大学毕业设计格式 (yzu_design)
- 智能标题层级识别
- 图表自动编号
- 列表格式处理
- 样式处理器
- 完整的异常处理体系
- 类型注解支持
- CLI 命令行接口
- 临时文件清理功能
- 全面的单元测试
- 完整的项目文档（README、用户指南、贡献指南、规则开发指南）

### Changed

- 简化 CI 配置，移除 lint 检查避免跨平台差异
- 优化文档结构，添加详细使用说明

### Fixed

- 修复四级标题编号显示问题（2.1.1.1 格式）
- 修复模块导入路径问题
- 修复 Ruff 代码风格警告

## 版本说明

- **Unreleased**: 未发布的更改，在 main 分支上开发中
- **X.Y.Z**: 版本号遵循语义化版本规范
  - X: 主版本号（不兼容的 API 更改）
  - Y: 次版本号（向后兼容的功能添加）
  - Z: 修订号（向后兼容的问题修复）

## 如何更新

```bash
# 使用 pip 更新到最新版本
pip install -U ruledoc

# 更新到特定版本
pip install ruledoc==2.0.0
```
